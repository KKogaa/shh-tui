# SHH TUI
A simple terminal chat client using Bubbletea UI.

## Installation

### Add config
```yaml
server:
  url: <server-ip>:8080
client:
  username: <username> 
  chatroom: <chatroom-name> 
  chat:
    width: <optional:width>
    height: <height>
  chatbox:
    width: <optional:width>
    height: <height>
```
